# 🎓 The Editorial Campus — Campus Internship & Placement Portal

A fully responsive, multi-page web application for managing college internships and placements.  
Built with pure **HTML · CSS · JavaScript** — no frameworks, no dependencies.

---

## 📄 Pages

| File | Page | Description |
|------|------|-------------|
| `index.html` | **Home Page** | Hero, features, journey, testimonials, CTA |
| `login.html` | **Login** | Role-based login (Student / Recruiter / Placement Cell) |
| `register.html` | **Register** | New account creation with role selection |
| `student-dashboard.html` | **Student Dashboard** | Profile, active applications, curated internships |
| `recruiter-dashboard.html` | **Recruiter Dashboard** | Stats, listings, candidates, interviews |
| `opportunities.html` | **Internship Listings** | Filter, browse, and apply for internships |
| `interview-prep.html` | **Interview Prep** | Technical, Aptitude, and HR video preparation |

---

## 🚀 How to Run Locally

1. Clone or download this repository
2. Open `index.html` in any browser
3. Navigate between pages using the navbar and buttons

No server setup required — it's fully static HTML.

---

## 🌐 How to Deploy on GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to **main branch / root**
4. Your site will be live at `https://yourusername.github.io/repository-name`
